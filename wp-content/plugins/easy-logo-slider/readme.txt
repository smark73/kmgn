== Description ==

Easy Logo Slider is plugin that helps users to upload the logos of clients, partners, and affiliates along with title, short description and website URL. For front-end display there is option to show or hide title and description and also will be the features like linking images, opening in new window.  Embed in any post/page using short-code <code>[jw_easy_logo slider_name="Slider Name"]</code> or you can even use widgets called Easy Logo Slider.

= Features =
* Simple and light weight
* Fully responsive
* Easy to use
* Pause on Hover
* Clean and Nice user interface
* Ability to add links to each logo if you want
* Auto-slide
* Create more than one logo slider with different logos and use different settings
* Can be add title, and descriptions and has option to show or hide in front-end

[Click here](http://plugins.jwthemes.com/easy-logo-slider-plugin-wordpress/) to see Easy Logo Slider in live action.

== Installation ==

1. Upload the `Easy Logo Slider` folder to the `/wp-content/plugins/` directory
1. Activate Easy Logo Slider plugin through the 'Plugins' menu in WordPress
1. Configure the plugin by going to the `Easy Logo Slider` tab that appears in your admin menu
1. Add to any page using short-code `["jw_easy_logo slider_name="Slider Name"]` or you can use widgets of `Easy Logo Slider`

== Frequently Asked Questions ==

= How can I use this in a widget? =

You can see the widgets name and if you drag that to widget location then you can find to select the slider name in drop-down from and select the slider name you want to use.
